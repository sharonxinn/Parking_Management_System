# Parking_Management_System
# Project Description 
Developed a file-based parking management system using Java to manage vehicle entry, exit, and parking records through a structured and persistent data storage approach.

# Key Features
-Built a console-based system to handle parking operations
-Implemented vehicle check-in and check-out functionality
-Managed parking records using a file-based storage system (read/write operations)
-Stored and tracked:
-Vehicle details (plate number, type)
-Entry and exit time
-Parking duration and fees
-Designed logic to calculate parking charges based on duration

# Tools & Concepts
-Java
-Object-Oriented Programming (OOP)
